# 木婦穴

**穴號**：圖5-5　**部位**：五五部位　腿前側穴位

## 董師原文
- **部位**：在足第二趾中節正中央外開三分。
- **主治**：婦科赤白帶下、月經不調、經痛、子宮炎、輸卵管 不通。
- **取穴**：當第二趾第二節正中央向外開三分是穴。
- **手術**：針深二分至四分，貼趾骨下針（用細毫針，粗針痛 苦）。

## 詮解發揮
### 維傑新用 / 董楊思維
黃帶、陰癢、尿道炎。 圖5-5 木婦穴 131

### 解說及發揮
1. 本穴治婦科病赤白帶極有效驗。 2. 此穴是腳上的婦科穴，本穴主治以婦科病為主。本穴在胃經，取名為木， 主治肝脾不和及肝膽溼熱之婦科病尤效。治療婦科赤白帶下、子宮炎。此外還 治月經不調、經痛、輸卵管不通。若針手不便時可針此，或與手婦科穴輪流應用。 3. 本穴基於“足軀順對”對應陰部，故治婦科子宮陰道病。亦可治尿道炎。 4. 本穴近於菜穴位置，榮主熱症，本穴治療黃帶、陰癢及尿道炎甚好。 5. 此穴雖治經痛，但餘以門金穴代之，尤有療效，且較不痛。 6. 董師日：貼趾骨下針（用細毫針，粗針痛苦）。此因為董師用針為26號或 28號針較粗所以然，現在一般刺此穴用30號或32號針，則未聞病人喊痛。 董師針足一般不脫襪子，針刺此穴則脫襪。 總按： 1.火包穴、海豹穴、木婦穴皆不超過腳趾與腳掌之連結紋，皆在第一節本 節，都屬井穴範圍，皆可治婦科病。火包穴、海豹穴在腳趾陰面尚可治疝氣。 2. 火包穴（胃經陰面）在足第二趾底第二道橫紋正中央。治難產、胎衣不 下、小腸疝氣。 3. 海豹穴（脾經）在大趾之內側，本節正中央，在隱白之後，大都之前，大趾 本節中央之黑白肉際。當大趾之內側，大趾本節正中央部（腳趾甲後）是穴。治 婦科陰道炎。 4. 木婦穴（胃經陽面）在足第二趾中節正中央外開三分。治婦科赤白帶下、 月經不調、經痛、子宮炎、輸卵管不通。 主要原理還是太極對應，在足太極中，腳趾端及井穴對應陰竅，腳趾對應少 腹，這三個穴皆對應於陰竅及少腹之間。 六六部位 總 論 一、董氏奇穴與五輸穴之五行屬性 在十四經穴中只有五輸穴有五行屬性，但這些五行屬性也影響著許多穴位 的五行屬性。五輸穴的五行屬性，陰經與陽經的配合次序是不同的，其和臨床應 用的關係很大，必須熟記，《難經•六十六難》說：“陰井木，陽井金，陰滎火，陽榮 水，陰輸土，陽輸木，陰經金，陽經火，陰合水，陽合土。”就是說陰經井木，依次為 榮火、輸土、經金、合水；陽經井金，依次為滎水、輸木、經火、合土。 五輸穴五行屬性對董氏奇穴的穴位定性取名很有影響，認識十四經五輸穴 之屬性及位置，有助於探索董氏奇穴之五行屬性，以便發揮擴充套件董氏奇穴之應 用，詳見楊維傑著《董氏奇穴原理解構》之“五行思路在董氏奇穴之擴充套件應用”。 董氏奇穴位於肘膝以下的穴位與五輸穴有密切關係。董氏奇穴在大腿的穴 位與合穴也有一定的聯絡。掌握五輸穴應用的空間及時間思路，能將董氏奇穴 應用得更好。 二、五輸主治所在及應用綱要 維傑認為五輸穴之主治最主要的為《靈樞經•順氣一日分為四時》所說：“病 133 在藏者取之井，病變於色者取之菜，病時間時甚者取之輸，病變於音者取之經，經滿 而血者病在胃，及飲食不節得病者取之於合。”及《靈樞經•邪氣臟腑病形》篇說 的：“滎輸治外經，合治內府。”以及《難經•六十八難》說的：“井主心下滿，滎主身 熱，輸主體重節痛，經主喘咳寒熱，合主逆氣而洩，此五臟六腑井滎輸經合所主病也。” 掌握這兩條就能活用五輸穴，也就能根據五輸穴應用原則活用位於肘膝以下的穴位。 三、“同氣相求” “同氣相求”療法是維傑對五輸穴最重要、最實用的用法思路，五輸穴透過 五行與臟腑有著相應的治療關係，這就是所謂的“同氣相求”。 同氣相求法又可分為相生、相剋、相應、相通及真五行等五類。但就單穴一 針療法而言最重要的是“相應”，下面只針對“相應”加以說明。 相應是同氣相求療法的中心用法，應用時以本經病為主經，旁及他經病變則 在本經找與其相應之穴，換言之以本經病主，寓有他髒病機參與者，可取本經 之五行相應穴。 由於五行屬性之聯絡，穴性屬木者，都能治療該經與肝及風及筋有關之疾 病。穴屬火，都能治療該經與心及火有關之疾病。穴屬土，都能治療該經與脾及 溼及肉有關之疾病。其他依此類推。 分論 120. 六六部位總圖見圖6-1。 • -水相穴 -水仙穴 水曲穴六完穴一 木鬥穴 -木留穴 門金穴 火主穴 -火硬穴 火散穴 火連穴 火菊穴 水晶穴 花骨二穴 花骨四穴 花骨三穴 >花骨一穴 圖6-1 六六部位總圖
