# 後枝穴

**穴號**：圖4-6　**部位**：四四部位　後臂穴位

## 董師原文
- **部位**：當肩中與肘之直線上，距富頂穴一寸，離肘橫紋八寸。
- **主治**：血壓高、頭暈、頭痛、殺菌、皮膚 病、血管硬化。
- **取穴**：手臂下垂，在後臂肱骨之外側， 距富頂穴一寸處是穴。
- **手術**：針深三分至七分。 運用：富頂、後枝兩穴同時下針，可治 頸項疼痛扭轉不靈及面部麻痺。

## 詮解發揮
### 維傑新用 / 董楊思維
本穴主治血壓高、頭暈、頭痛，實則皆與肝腎陰虛有關。滋陰補腎為根本療法。 富頂、後枝兩穴皆能治肝及調整血液迴圈，皆能治肝之血壓高、頭暈、頭痛、疲勞等。兩穴同時下針，可治頸項疼痛、扭轉不靈及面部麻痺。頸項疼痛、扭轉不靈多與血壓高有關。

### 解說及發揮
1.本穴主治血壓高、頭暈、頭痛，實則皆與肝腎陰虛有關。滋陰補腎為根本療法。 2. 富頂、後枝兩穴皆能治肝及調整血液迴圈，皆能治肝之血壓高、頭暈、頭 痛、疲勞等。兩穴同時下針，可治頸項疼痛、扭轉不靈及面部麻痺。頸項疼痛、扭 轉不靈多與血壓高有關。 3. 上臂距肘自4.5~8寸包括三焦經富頂、後枝，小腸經支通、落通、下曲，皆 治高血壓、頭暈、血管硬化。可見董氏奇穴系區位取穴及聯合針法。 4. 能殺菌、治皮膚病與分枝上穴、下穴接近，治療類近。
